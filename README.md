# timebox-app

iPadを中心に、PCブラウザでも利用できる個人用タイムボックスWebアプリです。

ポモドーロ専用ではなく、作業名と時間を自由に設定し、その日の予定を「時刻指定の簡易時刻表」と「時刻未指定のタスク一覧」で管理します。

## 開発方針

- HTML / CSS / JavaScript
- iPad Safari対応
- ホーム画面追加を想定
- PCブラウザ対応
- サーバー・ログインなし
- IndexedDB中心の端末内保存
- GitHub Pagesでの公開を予定
- PWA・オフライン対応は第2段階で追加予定

## 重要な設計方針

iPadでは、画面消灯中やバックグラウンド中にJavaScriptの定期処理が停止・遅延する可能性があります。

そのため、タイマーは単純な1秒減算だけに依存せず、開始時刻と終了予定時刻を保存し、復帰時に現在時刻との差から残り時間を再計算します。

## 実装段階

### 第1段階：基本版

予定登録、タイマー、履歴、IndexedDB保存、再読み込み・復帰時の時刻再計算など、タイムボックス管理の中核を実装します。

### 第2段階：拡張版

通知、終了音、PWA、オフライン、画面消灯防止、ドラッグ並べ替え、検索、JSONバックアップ、履歴編集、更新検出、GitHub Pages公開を追加します。

詳細仕様は [`TIMEBOX_SPEC.md`](./TIMEBOX_SPEC.md) を参照してください。

## 開発ルール

- 作業ブランチ：`codex-work`
- `main`へ直接コミット・プッシュしない
- force pushしない
- ユーザー確認前にマージしない
- 関係ファイルだけをコミットする
- テスト成功後に `origin/codex-work` へプッシュする

詳細は [`AGENTS.md`](./AGENTS.md) を参照してください。
